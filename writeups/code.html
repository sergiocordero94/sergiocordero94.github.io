HTB: Code

Write-up por Sergio Cordero

📝 Descripción

Este reto consiste en explotar una vulnerabilidad en HTTP (Gunicorn 20.0.4) para obtener acceso inicial al sistema Linux. Luego se realiza post-explotación para obtener privilegios y acceder al usuario Root.

🔧 Herramientas usadas

  • Nmap
  • Python
  • HashCat
  • NetCat

🚩 Resultado

Se obtuvo acceso con privilegios de ROOT y se recuperaron las flags de user y root.
