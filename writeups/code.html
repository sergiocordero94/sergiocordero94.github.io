HTB: Code

Write-up por Sergio Cordero

📝 Descripción

Este reto consiste en explotar una vulnerabilidad en SMB (EternalBlue) para obtener acceso inicial al sistema Windows XP SP3. Luego se realiza post-explotación básica para obtener información del sistema.

🔧 Herramientas usadas

  • Nmap
  • Metasploit
  • Searchsploit

🚩 Resultado

Se obtuvo acceso con privilegios de SYSTEM y se recuperaron las flags de user y root.
